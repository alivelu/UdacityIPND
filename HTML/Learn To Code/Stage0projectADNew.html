Basics of Web and HTML

The World Wide Web

World wide web is a collection of HTML documents.These documents are made up of something called HTML, which stands for HyperTextMarkup Language.HTML glues all sorts of files like documents, images, videos etc.The links between these pages are called Hyper Links.Web was invented in 1990.Main type of document on the web is HTML.All of these things/files like Microsoft word, PDF, Plain Text, music, videos, images exist on the web.

The major pieces are:
  • Person to operate the computer (you/me)
  • Computer/Computer running a web broswer like Mozilla Firefox/ Internet Explorer/ Chrome etc.
  • The Internet (Which is the world's largest computer network. The internet uses HTTP to communicate with you, your broswer and servers.
  • Servers

Browser is a program that runs on computer to display files found on the web.

Generally the browser makes requests via the internet to servers.

These requests are using a protocol called http and the servers respond with the files that the browser displays.

Servers are just like computers that host the files that makeup the web.

The protocol HTTP is a very simple protocol that your browser uses to communicate with these servers.

HTML means Hyper Text Markup Language

HTML is made up of :
  1. Text content i.e what you see
  2. Mark up i.e what the content look's like and how it's arranged
  3. References to other documents ex: images , videos
  4. It can be made up of Links to other pages
HTML Mark up

HTML markup is made up of things called tags.

Tag is structured like this:

Opening Tag Name.....The Tag Contents(which can be actualy tags or other tags)...... Closing Tag Name

The whole thing mentioned above is called as an element.

b-Tag means Bold tag

Structure is like this

opening bold tag .... Contents..... closing bold tag

Note: The Phrase in between bold tag appears on browser in bold letters

em-Tag means Emphasis tag (Italics)

Structure is like this

opening em tag .... Contents..... closing em tag

Note: The Phrase in between em tag appears on browser in bold and italicized letters

When you forget the closing em tag, Everything after the opening em tag is italicized.
