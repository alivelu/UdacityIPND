Basics of Web and HTML

The World Wide Web

World wide web is a collection of HTML documents.These documents are made up of something called HTML, which stands for HyperTextMarkup Language.HTML glues all sorts of files like documents, images, videos etc.The links between these pages are called Hyper Links.Web was invented in 1990.Main type of document on the web is HTML.All of these things/files like Microsoft word, PDF, Plain Text, music, videos, images exist on the web.

The major pieces are:
  • Person to operate the Computer (you/me).
  • Computer/Computer running a Web broswer like Mozilla Firefox/ Internet Explorer/ Chrome etc.
  • The Internet (Which is the world's largest computer network. The internet uses HTTP to communicate with you, your broswer and servers.
  • Servers

Browser is a program that runs on computer to display files found on the web. Generally the browser makes requests via the internet to servers. These requests are using a protocol called http and the servers respond with the files that the browser displays. Servers are just like computers that host the files that makeup the web. The protocol HTTP is a very simple protocol that your browser uses to communicate with these servers.

About HTML


HTML means Hyper Text Markup Language.

HTML is made up of :



  1. Text content i.e what you see
  2. Mark up i.e what the content look's like and how it's arranged
  3. References to other documents ex: images , videos
  4. It can be made up of Links to other pages

HTML Mark up

HTML markup is made up of things called tags.

About Tags

Tag is structured like this:
<NAME> Contents </NAME>
Opening Tag Name.....The Tag Contents(which can be actualy tags or other tags)...... Closing Tag Name
The whole thing mentioned above is called as an element.

Bold Tag

b-Tag means Bold tag.Structure is like this:
<b> contents </b>
opening bold tag .... Contents..... closing bold tag
Example:
HTML is<b>reasonably straightforward</b>
If you open the above code in browser it displays as below:
HTML is reasonably straightforward
Note: The Phrase in between bold tag appears on browser in bold letters.

Emphasis Tag

em-Tag means Emphasis tag (Italics).Structure is like this:
<em> contents </em>
opening em tag .... Contents..... closing em tag
Example:HTML is <em>reasonably straightforward</em>
If you open the above code in browser it displays as below:
Example:HTML is reasonably straightforward
Note: The Phrase in between em tag appears on browser in bold and italicized letters.
When you forget the closing em tag, Everything after the opening em tag is italicized.
HTML is <em><b>reasonably straightforward.</b>HTML is reasonably straightforward.
Note:If you open the above code in browser The Phrase after em tag is completely italicized

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them."br" tag is called as inline and it's just ending a line.
Inline elements are just text.
Ex: b tag, img tag, em tag.
"p" tag is block i.e it creates invisible box around text contains height and width.
Example: p tag

HTML-CSS-DOM

HTML : HyperText Markup Language
The standard markup language used to create web pages.
CSS : Cascading Style Sheets
Style sheet language used for describing the look and formatting of a document written in a markup language.
DOM : Document Object Model
A cross-platform and language-independent convention for representing and interacting with objects(Markup languages) in HTML . The nodes of every document are organized in a tree structure, called the DOM tree.

  • Html has specific syntax and rules.
  • Basic word in html language is tag.
  • The Browser turns html tags into elements that form a tree.
  • It knows how to do that because of document model DOM.
  • DOM is a standard convention for representing and interacting in HTML.
  • So the tree in developer tools in chrome browser is dom tree the browser built from an HTML ducument.
  • Each html tag creates an element in DOM that the browser uses to display the page.

An element is created from starting tag to ending tag and everything in between is content.
<tag> content </tag>
Syntax: opening tag content closing tag
The content can be empty, it can be text or element. Each tag has a name or type that defines what kind of element will be created. I.e will it be a paragraph?, An image?, Or Just a document division..? Each tag also can have attributes with values.
Syntax: <h1 class = "article"> </h1> For example above we have an h1 tag with a class attribute and value of the attribute is article. So the document written in HTML defines the structure and content of the page.
